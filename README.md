# Kontingentet 2026 — München

Statisk nettside klar for GitHub-opplasting. Ingen byggesteg eller avhengigheter kreves.

## Innhold

- nedtelling til avreise
- reiseramme basert på Alternativ A
- pakkeliste med lokal lagring
- bierliste med poeng
- challenges
- tekstbasert RPG med flere avslutninger
- klassisk 70-talls østtysk plakatestetikk

## Filer

- `index.html` – selve appen
- `styles.css` – layout og tema
- `app.js` – logikk, localStorage og RPG
- `assets/favicon.svg` – ikon

## Publisering

### GitHub Pages

1. Opprett et nytt repository
2. Last opp alle filene i denne mappen
3. Push til `main`
4. Aktiver GitHub Pages fra repository settings

Nettsiden fungerer direkte som statisk side.

## Lokalt

Åpne `index.html` i nettleseren, eller kjør en enkel lokal server.
